# Cheat Sheet

## fugitive
- :Gstatus  : git status
- :Gread    : 特定コミットの内容を表示(buff) opt: - 最新コミット, ^ 一つ前のコミット ~n n前のコミット(n=数値)
- :Gwrite   : git add
- :Gcommit  : git commit
- :Gpush    : git push
- :Gfetch   : git fetch
- :Gmerge   : git merge
- :Gpull    : git pull
- :Glog     : git log
- :Gdiff    : git diff
- :Gblame   : ファイルの変更履歴確認
